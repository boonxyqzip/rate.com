# 🎯 티어메이커 (Tier Maker)

웹 기반 티어메이커 사이트입니다. 아이템을 드래그 앤 드롭하여 티어에 배치할 수 있습니다.

## ✨ 주요 기능

- **티어 시스템**: S, A, B, C, D, F 티어 지원
- **드래그 앤 드롭**: 직관적인 드래그 앤 드롭 인터페이스
- **아이템 관리**: 
  - 텍스트 이름으로 아이템 추가
  - 이미지 업로드 지원
  - 개별 아이템 삭제
- **데이터 저장**: 로컬 스토리지에 자동 저장
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 아이템 이름을 입력하고 (선택적으로 이미지 업로드) "아이템 추가" 버튼 클릭
3. 아이템을 드래그하여 원하는 티어에 배치
4. "저장" 버튼으로 데이터 저장, "불러오기"로 불러오기

## 📁 파일 구조

```

├── styles.css      # 스타일시트

```
